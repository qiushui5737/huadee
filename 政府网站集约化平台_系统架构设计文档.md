## 1. 架构设计总则

### 1.1 架构设计原则

| 原则 | 说明 |
|------|------|
| **前后端分离** | 前端Vue3/后端Spring Boot 3通过RESTful API + SSE通信 |
| **微服务化** | 按业务域拆分25个模块，每个模块可独立开发部署 |
| **DDD领域驱动** | 每个模块有明确的限界上下文(Bounded Context) |
| **一人全栈** | 每人负责≥5个模块的完整前后端流程，减少跨人沟通成本 |
| **API优先** | 先定接口契约(Mock)，前后端并行开发 |
| **AI驱动** | 搜索与问答场景深度集成Spring AI |

### 1.2 技术栈选型

| 层次 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **前端框架** | Vue 3 + TypeScript | 3.4+ | C端用户界面 + B端管理后台 |

```java
@RestController
@RequestMapping("/api/ai/chat")
public class ChatController {

    @Autowired private ChatService chatService;

    // SSE流式问答
    @PostMapping(value = "/stream", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public SseEmitter chatStream(@RequestBody ChatRequestDTO request,
                                 @RequestHeader("Authorization") String token) {
        // 1. 验证用户身份
        // 2. 创建SSEEmitter (超时60s)
        SseEmitter emitter = new SseEmitter(60000L);

        // 3. 异步执行流式问答
        chatService.streamChat(request, emitter, userId);

        return emitter;
    }
}

@Service
public class ChatService {

    private final ChatClient chatClient;              // Spring AI
    private final RagService ragService;             // RAG检索
    private final SensitiveFilterService sensitive;   // A4敏感词
    private final ChatMemoryService memoryService;    // A5会话记忆

    public void streamChat(ChatRequestDTO request, SseEmitter emitter, String userId) {
        String sessionId = request.getSessionId();
        String userMessage = request.getMessage();

        // Step1: 输入端敏感词过滤
        if (sensitive.isSevere(userMessage)) {
            emitter.send("抱歉，您的提问包含敏感内容，请修改后重试。");
            emitter.complete();
            return;
        }
        String filteredInput = sensitive.filter(userMessage);

        // Step2: RAG检索相关政务文档
        List<Document> docs = ragService.retrieve(filteredInput, 5);
        String context = ragService.buildContext(docs);

        // Step3: 获取历史记忆
        List<Message> history = memoryService.getHistory(sessionId, 10);

        // Step4: 构建Prompt并流式调用
        String systemPrompt = """
            你是一名专业的政务助手，负责回答群众关于政务办事、政策法规等问题。
            请基于以下政务知识库内容回答，如果知识库中没有相关信息，请如实告知。
            
            【政务知识库】
            %s
            """.formatted(context);

        Prompt prompt = new Prompt(List.of(
            new SystemMessage(systemPrompt),
            // 历史消息
            ...history,
            new UserMessage(filteredInput)
        ));

        // Step5: 流式输出
        chatClient.stream(prompt)
            .doOnNext(response -> {
                String content = response.getResult().getOutput().getContent();
                // 输出端敏感词过滤
                content = sensitive.filterOutput(content);
                emitter.send(SseEmitter.event().data(content));
            })
            .doOnComplete(() -> {
                // 保存完整会话记录
                memoryService.saveMessage(sessionId, userId, userMessage, fullResponse);
                emitter.complete();
            })
            .doOnError(emitter::completeWithError)
            .subscribe();
    }
}
```

```java
public class RedisChatMemory implements ChatMemory {

    private final RedisTemplate<String, Object> redisTemplate;
    private static final int MAX_MESSAGES = 20; // 最近10轮=20条消息
    private static final long TTL_MINUTES = 30;  // 30分钟过期

    @Override
    public void add(String sessionId, String role, String content) {
        String key = "chat:memory:" + sessionId;
        redisTemplate.opsForList().rightPush(key, new ChatMessage(role, content));
        // 只保留最近20条
        redisTemplate.opsForList().trim(key, -MAX_MESSAGES, -1);
        redisTemplate.expire(key, TTL_MINUTES, TimeUnit.MINUTES);
    }

    @Override
    public List<Message> getHistory(String sessionId, int lastN) {
        String key = "chat:memory:" + sessionId;
        List<Object> raw = redisTemplate.opsForList().range(key, -lastN, -1);
        return raw.stream()
            .map(o -> (ChatMessage) o)
            .map(m -> "user".equals(m.getRole()) ?
                new UserMessage(m.getContent()) : new AssistantMessage(m.getContent()))
            .collect(Collectors.toList());
    }
```

```java
@Service
public class OverdueMonitorService {

    // 定时扫描超时留言 (每30分钟)
    @Scheduled(fixedRate = 1800000)
    public void scanOverdue() {
        List<InteractionMessage> overdue = messageMapper.selectList(
            new LambdaQueryWrapper<InteractionMessage>()
                .lt(InteractionMessage::getReplyDeadline, LocalDateTime.now())
                .notIn(InteractionMessage::getStatus, List.of("REPLIED", "CLOSED"))
        );

        for (Message msg : overdue) {
            // 更新状态为超时
            msg.setStatus("OVERDUE");
            messageMapper.updateById(msg);

            // 上报督办通知
            notificationService.sendUrgentNotification(
                msg.getDeptCode(),
                "留言超时未回复: " + msg.getTitle()
            );
        }
    }
}
```

```java
@Service
public class DeadlineMonitorService {

    // 时限配置
    private static final int TOTAL_DEADLINE_DAYS = 20;    // 总时限20工作日
    private static final int MAX_EXTEND_DAYS = 20;        // 最多延长20日
    private static final int WARNING_DAYS = 5;           // 剩余5天预警

    // 定时检查时限 (每天9:00)
    @Scheduled(cron = "0 0 9 * * ?")
    public void checkDeadlines() {
        List<DisclosureApplication> list = applicationMapper.selectActiveApplications();

        for (DisclosureApplication app : list) {
            int remainingDays = calculateRemainingDays(app);

            if (remainingDays < 0) {
                // 超时
                app.setStatus("OVERDUE");
                notificationService.sendUrgent(app.getAssignedDept(),
                    "依申请公开超时: " + app.getApplyNo());
            } else if (remainingDays <= WARNING_DAYS) {
                // 预警
                notificationService.sendWarning(app.getAssignedDept(),
                    "依申请公开即将到期: " + app.getApplyNo() + " 剩余" + remainingDays + "天");
            }
        }
    }
}
```

```java
@Service
public class IndexSyncService {

    // 增量同步: 监听MQ消息
    @RabbitListener(queues = "index.sync.queue")
    public void onContentChanged(IndexSyncMessage msg) {
        String action = msg.getAction();   // CREATE/UPDATE/DELETE
        String contentType = msg.getType(); // NEWS/POLICY/NOTICE/GUIDE
        Long contentId = msg.getContentId();

        switch (action) {
            case "CREATE":
            case "UPDATE":
                // 从CMS(D2)获取内容 → 构建索引文档 → 写入ES
                ContentArticle article = contentService.getById(contentId);
                SearchDocument doc = buildSearchDocument(article);
                esRepository.save(doc);
                break;
            case "DELETE":
                esRepository.deleteById(contentId.toString());
                break;
        }
    }

    // 全量重建索引
    public void rebuildAll() {
        // 1. 删除旧索引
        esRepository.deleteIndex("gov_search_index");
        // 2. 创建新索引(mapping)
        esRepository.createIndex("gov_search_index", getIndexMapping());
        // 3. 从各数据源批量拉取 → 批量写入ES
        List<ContentArticle> all = contentService.listAll();
        List<SearchDocument> docs = all.stream()
            .map(this::buildSearchDocument)
            .collect(Collectors.toList());
        esRepository.bulkSave(docs);
    }
}
```

```javascript
      });
    };

    // 按钮点击上报

      this.send({
        event: 'BUTTON_CLICK',
        buttonId,
        pageUrl,
        timestamp: Date.now()
      });
    };
  },

  // 批量发送(30秒攒一批)
  send(data) {
    if (!this.buffer) this.buffer = [];
    this.buffer.push(data);
    if (this.buffer.length >= 10) {
      this.flush();
    }
  },

  flush() {
    navigator.sendBeacon('/api/track/batch', JSON.stringify(this.buffer));
    this.buffer = [];
  }
};
```

```java
@Service
public class ScoreCalculateService {

    // 按周期自动计算各部门绩效评分
    @Scheduled(cron = "0 0 3 1 * ?")  // 每月1日3:00执行
    public void calculateMonthlyScore() {
        Long periodId = getCurrentPeriodId();
        List<Long> deptIds = deptService.listAllDeptIds();

        for (Long deptId : deptIds) {
            List<PerformanceIndicator> indicators =
                indicatorConfigService.getIndicators(deptId);

            BigDecimal totalScore = BigDecimal.ZERO;
            for (Indicator indicator : indicators) {
                // 从E2统计服务获取指标原始数据
                BigDecimal rawValue = statsService.getIndicatorValue(
                    deptId, indicator.getCode(), periodId);

                // 按评分规则计算得分
                BigDecimal score = calculateScore(indicator, rawValue);

                // 保存单项评分
                saveScoreRecord(periodId, deptId, indicator.getId(), score);
                totalScore = totalScore.add(score);
            }

            // 保存总分
            saveTotalScore(periodId, deptId, totalScore);
        }

        // 计算排名
        rankingService.calculateRanking(periodId);
    }

    private BigDecimal calculateScore(Indicator indicator, BigDecimal rawValue) {
        // 评分规则: 线性/阶梯/公式
        switch (indicator.getScoreRule()) {
            case LINEAR:  // 线性: 得分 = 原始值 × 权重 × 100
                return rawValue.multiply(indicator.getWeight());
            case STEP:    // 阶梯: 不同区间不同得分
                return calculateStepScore(indicator, rawValue);
            case FORMULA: // 自定义公式
                return calculateFormulaScore(indicator, rawValue);
        }
    }
}
```

#### 8.5.2 消息分发流程

```java
@Service
public class NotificationListener {

    // 监听所有服务的通知消息
    @RabbitListener(queues = "notification.queue")
    public void onNotification(NotificationMessage msg) {
        // 1. 根据消息类型获取模板
        MessageTemplate template = templateService.get(msg.getType());

        // 2. 填充模板变量
        String content = templateService.render(template, msg.getVariables());

        // 3. 站内消息 (始终发送)
        inAppMessageService.send(msg.getUserId(), template.getTitle(), content);

        // 4. 根据用户偏好发送
        UserPreference pref = getUserPreference(msg.getUserId());
        if (pref.isSmsEnabled()) {
            smsService.send(msg.getPhone(), content);
        }
        if (pref.isWeChatEnabled()) {
            weChatNotifyService.sendTemplateMessage(msg.getOpenId(), msg.getType(), msg.getVariables());
        }
    }
}
```

## 12. 前后端联调与协作规范

### 12.1 接口先行原则

开发流程采用 **API-First** 原则，后端先输出接口文档(Mock)，前端基于Mock并行开发：

```
阶段1: 需求评审 → 接口设计 → 输出OpenAPI文档 (Day 1-2)
阶段2: 后端Mock接口 + 前端基于Mock开发 (Day 3-7)
阶段3: 后端实现真实接口 → 前端切换真实接口联调 (Day 8-12)
阶段4: 集成测试 → 联调修复 → 验收 (Day 13-15)
```

### 12.4 前端API封装规范

```typescript
// gov-shared/api/request.ts
import axios from 'axios'

const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE,
  timeout: 30000
})

// 请求拦截: 添加Token
request.interceptors.request.use(config => {
  const token = localStorage.getItem('access_token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

// 响应拦截: 统一处理
request.interceptors.response.use(
  response => {
    if (response.data.code === 200) return response.data
    if (response.data.code === 401) {
      // Token过期 → 刷新或跳转登录
      refreshTokenOrRedirect()
    }
    ElMessage.error(response.data.message)
    return Promise.reject(response.data)
  },
  error => {
    if (error.response?.status === 429) {
      ElMessage.warning('请求过于频繁，请稍后重试')
    }
    return Promise.reject(error)
  }
)

export default request
```
