# 统一数据库运行说明

## 统一数据库

- 数据库名：`gov_platform`
- 默认账号：`root`
- 默认密码：`050420`
- 统一建表脚本：`gov_platform_all_tables.sql`

## 成员第一次创建数据库

进入本目录：

```bash
cd gov-platform/sql
```

执行：

```bash
mysql -uroot -p050420 < gov_platform_all_tables.sql
```

如果不想把密码写在命令里，也可以执行：

```bash
mysql -uroot -p < gov_platform_all_tables.sql
```

然后按提示输入密码。

## 项目数据库配置

后端默认读取 `gov-platform/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: ${DB_URL:jdbc:mysql://localhost:3306/gov_platform?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai}
    username: ${DB_USERNAME:root}
    password: ${DB_PASSWORD:050420}
```

每个人本机密码不同也没关系，可以在本机环境变量里覆盖：

```bash
DB_USERNAME=root
DB_PASSWORD=自己的数据库密码
DB_URL=jdbc:mysql://localhost:3306/gov_platform?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
```

代码里不要写自己的个人密码，统一提交环境变量写法。

## 表数量

当前统一脚本包含 39 张表，覆盖 A/B/C/D/E 五个模块：

- A：智能搜索、语义搜索、智能问答、敏感词、会话审计
- B：办事事项、动态表单、审批流程、缴费、电子证照
- C：政民互动、留言督办、依申请公开、问卷调查
- D：部门子站、CMS、内容审核、信息公开目录
- E：数据采集、报表绩效、统一认证、消息中心
